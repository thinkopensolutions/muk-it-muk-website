MuK Gradient Option

Customize Background Option

MuK IT GmbH - www.mukit.at

Overview

Extends the Odoo Website Customize Background Color menu to give a selection of gradient backgrounds. The gradient background can be used like any other background color provided by Odoo.

Color Scheme

Customize

Change the color scheme by editing the less file through the HTML/CSS editor. The variables can be found in the variables.less file located in the static folder of the module.
